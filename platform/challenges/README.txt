Défis Stéganographie et Cryptographie
=====================================

Les fichiers sont générés au build de l'image Docker (voir Dockerfile).
Si tu exécutes la plateforme sans Docker, tu peux créer les défis manuellement :

Stéganographie :
- image_stego.jpg : image avec fichier caché (steghide), mot de passe : labstego

Cryptographie :
- flag_b64.txt : flag encodé en Base64
- flag.enc : fichier chiffré OpenSSL (mot de passe : labcrypto)
